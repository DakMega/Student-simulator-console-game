This is my first game, which using C++ language. 
Эта простая игра кратко повествует об основных эвентах в жизни студента в юмористической форме.
Интерфейс напоминает старые консоли пк времен 90-х.
Игра написана на инглийском языке, но со вставками транслита, чтобы передать забавную атмосферу студенческой жизни. 
В игре реализованы основные элементы ООП, динамическая память и т.д.
Исполняемый файл находится в папке Debug.
